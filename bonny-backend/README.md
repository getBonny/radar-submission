## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

```bash
docker run --name postgres -e POSTGRES_PASSWORD=mysecretpassword -p5432:5432 -d postgres
```
## Generate Migration Files
`NODE_ENV` doesn't seem to work here, so you have to set the environment variables inline
```bash

cross-env DB_USER=postgres DB_PASSWORD=mysecretpassword npm run migration:generate src/migration
```

## Running the app

```bash
# local - you may need to add an env-file in config folder and add an script entry in package.json
$ npm run start:<yourname>

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running in dev container

```bash
source .env.local && npm run start:dev
```

### Build Image
```bash
gcloud builds submit --tag gcr.io/locards-bonny-test/bonny-backend
```

### Deploy into Container
```bash
gcloud run deploy --service-account='bonny-test-backend@locards-bonny-test.iam.gserviceaccount.com' --env-vars-file=.env.yaml --image='gcr.io/locards-bonny-test/bonny-backend' --region='europe-west3' --add-cloudsql-instances='locards-bonny-test:europe-west3:bonny-test-db'
```
